## üéì Key Concepts

### 1. Three-Section Layout
Toolbar divided into functional groups:
- **Actions** - Export & Refresh
- **Search & View** - Search input + View toggle
- **Stats** - Card-based statistics

### 2. Card-Based Stats
Statistics displayed as cards with:
- Color-coded icons
- Category labels
- Count + quantity
- Hover effects
- Update animations

### 3. Progressive Disclosure
Content adapts to screen size:
- Desktop: Everything visible
- Tablet: Stats collapsible
- Mobile: Compact + collapsed

### 4. Event-Driven Communication
Components communicate via CustomEvents:
- `rk:refresh` - Refresh requested
- `rk:dataLoaded` - Data loaded
- `rk:toolbarReady` - Toolbar initialized

### 5. Accessibility First
Complete keyboard navigation:
- Tab order
- Shortcuts (Ctrl+R, Ctrl+F, Esc)
- ARIA labels
- Screen reader support
