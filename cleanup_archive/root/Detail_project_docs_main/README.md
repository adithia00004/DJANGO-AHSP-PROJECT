# Folder `Detail_project_docs_main`

Dokumen di folder ini berisi referensi modul inti selain Jadwal Kegiatan.

| File | Modul | Isi Utama |
| --- | --- | --- |
| `LIST_PEKERJAAN_DOCUMENTATION.md` | List Pekerjaan | Arsitektur drag-and-drop, event handling, testing, dan best practice reorder pekerjaan. |
| `VOLUME_PEKERJAAN_DOCUMENTATION.md` | Volume Pekerjaan | Mekanisme input volume, integrasi API, serta edge case validasi. |
| `TEMPLATE_AHSP_DOCUMENTATION.md` | Template AHSP | Deskripsi modul Template AHSP, dependency referensi, dan SOP sinkronisasi data. |
| `HARGA_ITEMS_DOCUMENTATION.md` | Harga Items | Penjelasan pengelolaan harga satuan, struktur UI, serta titik integrasi dengan sumber data. |

Setiap dokumen memuat flow UI/UX, API yang dipakai, serta checklist testing yang relevan. Gunakan tabel ini bersama `DOCUMENTATION_OVERVIEW.md` untuk navigasi cepat.
